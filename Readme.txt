2) Описание выбранной идеи решения:
Пользователю предлагается возможность переходить по меню программы через консольный ввод. Запись и чтение информации происходит через txt файлы. Всё, что идёт после пробела, при вводе - игнорируется (исключение - сообщения, там записывается всё). Это сделано для исключения зацикливаний (в том числе бесконечных) программы.

Регистрация:
При регистрации нового пользователя необходимо дважды ввести пароль, даётся две попытки. Регистрация с уже имеющимся логином невозможна. Авторизация происходит автоматически после успешной регистрации. Данные пользователей хранятся в logfile.txt

Авторизация:
Тут всё просто. Если логин и пароль совпадают, то входим в личный кабинет.

Приватный чат:
В начале необходимо выбрать пользователя, с которым вы хотите вести чат. Рядом с логином указывается кол-во новых сообщений. При выборе пользователя по умолчанию выводится 10 последних сообщений. Командой /w"число" можно вывести выбранное пользователем кол-во сообщений (Пример: /w8), не более 999 сообщний. После отправки сообщения чат автоматически обновляется. Запись происходит в messages"Имя пользователя1""Имя пользователя2".txt

Общий чат:
Аналогичен приватному чату. По умолчанию выводится 10 последних сообщений. Командой /w"число" можно вывести выбранное пользователем кол-во сообщений (Пример: /w17), не более 999 сообщений. После отправки сообщения чат автоматически обновляется. Запись происходит в messagesGlobal.txt
